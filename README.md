## Run Locally

Clone the project

```bash
  git clone https://github.com/JonnathanE/landing-page-002.git
```

Go to the project directory

```bash
  cd landing-page-002
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
